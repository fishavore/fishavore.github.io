SSIS Package Reader

SSIS Package Reader documents Microsoft SQL Server Integration Services (SSIS) package contents. It is a C# program that I created to help discover the contents of packages during SQL Server migration.

You can find the SSIS Package Reader program on my GitHub repository:

https://github.com/fishavore/SsisPackageReader

Requirement

To successfully read SSIS package contents, connectors, such as Microsoft Connectors v2.0 for Oracle and Teradata, that were used to create the package are required on the computer you want to run this program.

How to use SSIS Package Reader

  1. Clone or download SSIS Package Reader from the repository.
  2. Compile the program.
    Note: The latest project files were created with Microsoft Visual Studio 2012 and .NET Framework 4.5.
  3. Start SSIS Package Reader.
  4. On the File menu, click Open.
  5. In the Browse For Folder dialog box, choose a folder that contains SSIS package files, and then click OK. All contents based on the supported tasks are displayed.
  6. To search for a specific task, type the text that you want SSIS Package Reader to search for in the Search Term box, and then click Search.
  7. To display all contents again based on the supported tasks in the selected folder, click Search All.

Limitations

Currently, the following types of SSIS tasks are supported:

  • Data Flow Task
  • Execute SQL Task
