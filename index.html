SSIS Package Reader

SSIS Package Reader documents SSIS package contents. It is a program that I created to help discover the contents of packages during Microsoft SQL Server migration.

You can find the SSIS Package Reader program on my GitHub repository:

https://github.com/fishavore/SsisPackageReader

How to use SSIS Package Reader

  1. Start SSIS Package Reader.
    Note: Binary files are excluded from the project files.
  2. On the File menu, click Open.
  3. In the Browse For Folder dialog box, choose a folder that contains SSIS package files, and then click OK. All contents based on the supported tasks are displayed.
  4. To search for a specific task, type the text that you want SSIS Package Reader to search for in the Search Term box, and then click Search.
  5. To display all contents again based on the supported tasks in the selected folder, click Search All.
